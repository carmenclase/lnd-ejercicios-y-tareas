# Ejercicio 4. 
Generar un fichero HTML donde se resalten los productos con precio mayor a 500 en negrita `<b>`.

XML de entrada:
```
<products>
  <product>
    <name>Laptop</name>
    <price>1000</price>
  </product>
  <product>
    <name>Mouse</name>
    <price>25</price>
  </product>
</products>
```
Salida:
```
<p><b>Laptop</b> - $1000</p>
<p>Mouse - $25</p>
```
